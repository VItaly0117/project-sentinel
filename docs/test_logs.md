## 📊 Sentinel MVP: Stability Test Report (Day 1)

**Дата:** 22.04.2026

**Длительность теста:** ~1.5 часа

**Стенд:** MacOS, Python 3.12, Bybit API (Live Data)

**Режим:** `dry-run` (симуляция)

### ✅ Основные достижения:

1. **Uptime 100%:** Бот отработал без падений, утечек памяти или обрывов соединения. Circuit Breaker корректно обрабатывал мелкие сетевые заминки.
    
2. **Интеграция с Bybit:** Успешный парсинг 5-минутных свечей (Klines). Задержка обработки данных после закрытия свечи — менее 1 сек.
    
3. **Risk Engine (Virtual Equity):** Реализована и протестирована логика «виртуального кошелька». Бот корректно считает просадку и профит, не имея реальных средств на счету.
    
4. **Model Inference:** Модель `monster_v4_2.json` (XGBoost) успешно принимает фичи и выдает предсказания. Средний уровень уверенности на текущем рынке: `0.38 - 0.48`.
    

### 🛠 Исправленные баги:

- **API Key 10003:** Исправлена ошибка авторизации (переход с Demo на Live ключи).
    
- **Equity Panic:** Исправлена логика риск-менеджера — теперь в режиме `dry-run` используется `STARTING_BALANCE`, что предотвращает ложную блокировку из-за нулевого реального баланса.
